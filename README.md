# つぶやき帳 デモ

iPhoneから1000文字の投稿、写真10枚、4GBまでの動画選択、YouTubeリンク追加、編集、削除を試せる旧Twitter風の軽量タイムラインです。

投稿写真はタップで拡大できます。削除操作には画面内の確認ダイアログを使用します。

管理者は表示名、ユーザー名、紹介文、アイコン、プロフィールバナーを変更できます。

## サンプル用パスワード

閲覧用は `2525` です。管理者用パスワードでログインした場合だけ、投稿、編集、削除が表示されます。

## 起動

```bash
npm install
npm run dev
```

## 注意

このサンプルは操作確認用です。パスワード、投稿、写真はブラウザ内で処理されます。動画は選択と現在のセッション内での再生確認に対応します。4GB動画を端末間で共有する本番運用には、大容量ストレージと分割アップロードAPIを接続してください。
